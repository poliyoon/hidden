# Project Name: The Hidden (The Private Curator)

## 1. Project Definition
- **Concept:** HNWI(고액 자산가)를 위한 폐쇄형 여행 & 다이닝 큐레이션 플랫폼.
- **Core Value:** Scarcity(희소성), Privacy(프라이버시), Human-touch(사람 냄새 나는 기술).
- **Target Audience:** 구글 검색을 귀찮아하고, 검증된 소수의 정보를 원하는 리치 유저.

## 2. Design System & Tone
- **Visual Theme:** "Quiet Luxury"
- **Colors:**
  - Background: Deep Black (#121212) or Dark Charcoal (#1A1A1A)
  - Text: Off-White (#F5F5F5) for readability, Muted Gold (#D4AF37) for accents.
- **Typography:**
  - Headings: Serif fonts (e.g., Playfair Display, Cinzel) - 우아함 강조.
  - Body: Sans-serif fonts (e.g., Lato, Inter) - 가독성 강조.
- **UI Style:** Minimalist, Card-based layout, High-quality imagery, Generous whitespace.

## 3. Tech Stack (Antigravity Base)
- **Backend:** Django, Python 3.11+
- **Database:** PostgreSQL
- **Frontend:** Django Templates + Tailwind CSS (HTMX for interactivity preferred over React for SEO/Speed)
- **Auth:** Custom Invite-only system (No open sign-ups).

## 4. Key Features (MVP)
- **Deep Profiling:** AI Chat-based preference survey.
- **Black Book:** Database of curated, private venues.
- **Concierge Button:** Direct line to human manager (No automated booking API).
